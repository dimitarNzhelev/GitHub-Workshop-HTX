## Инструкции за пускане на проекта

1. Създаване на Python виртуална среда *(ако не използвате PyCharm)*
```console
$ python -m venv venv
```

2. Инсталиране на зависимостите
```console
$ pip install -r requirements.txt
```

3. Стартиране на сървъра
```console
$ python app.py
```
